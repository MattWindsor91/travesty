M is the monad over which we're bi-traversing.

val bi_map_m : ('l1, 'r1) t -> left:('l1 left -> 'l2 left M.t) -> right:('r1 right -> 'r2 right M.t) -> ('l2, 'r2) t M.t

bi_map_m c ~left ~right monadically traverses with left over every 'l1 left, and right over every 'r1 right, in c.
