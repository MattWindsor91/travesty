Module type Mappable_types.S1_container

S1_container is the signature of an arity-1 mappable container.

include S1
type 'a t

t is the type of the container to map over.

include Generic with type ('a, 'phantom) t := 'a t and type 'a elt := 'a

Generic refers to the container type as 'a t, and the element type as 'a elt; substitute t/elt (arity-0) or 'a t/'a (arity-1) accordingly below.

include Generic_types.Generic with type ('a, 'phantom) t := 'a t with type 'a elt := 'a
val map : 'a t -> f:('a -> 'b) -> 'b t

map c ~f maps f over every t in c.
