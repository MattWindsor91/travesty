Module Travesty__T_monad

Generic monad extensions.

T_monad contains a signature and functor for adding various extensions to Core-style monads.

Miscellaneous

module S2_to_S : functor (M : Base.Monad.S2) -> functor (B : Base.T) -> Base.Monad.S with type 'a S2_to_S.t := ('a, B.t) M.t

S2_to_S demotes an arity-2 monad M to an arity-1 one, fixing its second type to be B.t.
